# TDD_LRU_Cache_CS3318

This is a basic LRU Cache program implemented using LRU algorithm and Test Driven Development. 

TDD ensures every feature of the LRU emerges only in response to a failing test, improving correctness and preventing lengthy and unnecessary code. 

There is a detailed report in the repository which explains full implementation in detail. 

Tech Stack: Java, Maven (JUnit5 included as dependency) 

Instructions to download: 
1. Make sure to have IntelliJ IDEA or similar with Maven and JUnit5 compatibility.
2. Clone Repository.
3. Run tests.
